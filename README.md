# Explainable-Credit-Risk-Scoring-System

• Built an ML-based credit risk scoring model to predict loan default probability using customer financial and behavioral features
• Applied feature engineering, class imbalance handling, and ensemble models to improve predictive performance
• Used SHAP explainability to interpret model decisions and highlight key risk-driving factors
• Achieved high ROC-AUC with transparent, business-friendly decision insights

Technologies: Python, Scikit-learn, XGBoost, SHAP, Pandas, NumPy
